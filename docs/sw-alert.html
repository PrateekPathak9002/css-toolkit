To use the sw-alert class , first import jquery,swifty_js_toolkit, and swifty-css . <script src="https://code.jquery.com/jquery-2.1.0.js"</script> <script src="https://unpkg.com/swifty_js_toolkit@1.1.8/swifty.js"</script> <link rel="stylesheet" href="https://unpkg.com/swifty-css@1.2.3/swifty.css"> Now to use the alerts , we make a div for the alert. Its class should be sw-alert-type-size. type can be primary/secondary/danger/success.Size can be sm/md/full. 
<div class="sw-alert-primary-md"></div>
Now inside this , you should add the text you want to display on this alert. 
<div class="sw-alert-primary-md">
	 Hi! I'm a swifty alert !
</div>
Thats it for a simple non-dismissable alert.Continue reading for dismissing the alert.

Dismissing

To dismiss, simply add -dismissable to the end of the class. Along with that , add a div inside the main div that has &times as the content and class sw-alert-dismiss. 
<div class="sw-alert-primary-md-dismissable"> 

          Hi . You are reading a cool alert . 
 
          <div class="sw-alert-dismiss">&times;</div> 

</div>
Now, you may want to change the color of the background and text. You can do that using css , but also using swifty-css's way. 
<div class="sw-alert-primary-md-dismissable"> 

          Hi . You are reading a cool alert . 
 
          <div class="sw-alert-dismiss">&times; <!--{bg_color:'red' , color:'black'}--></div> 

</div>
